# Releases

Índice de carpetas por versión SemVer. Cada versión incluye:

- `release_vX.Y.Z.md` — notas (plantilla en [`../templates/release-template.md`](../templates/release-template.md))
- `checklist-pr.md` — registro de revisiones de la campaña
- `PULL_REQUEST.md` — referencia del PR de campaña (opcional)
