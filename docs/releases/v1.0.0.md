# SDD Kit — versión 1.0.0

> **Estado:** primera versión **estable** del kit. Lista para usar en proyectos reales con submódulo Git.

**Fecha de release:** 2026-06-11  
**Repositorio:** https://github.com/jcalistop/sdd-kit

---

## ¿Qué es el SDD Kit?

Una metodología y un conjunto de herramientas para desarrollar **con un plan antes de codear**.

1. Anotas la idea en un backlog.
2. Escribes un **spec** (documento corto): qué quieres, qué no incluye, cómo sabrás que está listo.
3. Implementas — tú o un **agente de IA** — siguiendo ese spec.
4. Revisas, publicas y archivas el trabajo.

Pensado para **equipos pequeños** y para quien trabaja **solo con ayuda de IA** (Cursor, Claude Code, Codex, Copilot).

---

## ¿Qué hay de nuevo respecto a la 0.1.0?

La **0.1.0** fue una pre-release de prueba (solo Laravel + Cursor). La **1.0.0** consolida todo lo aprendido:

| Área                     | Qué ganaste                                                                        |
| ------------------------ | ---------------------------------------------------------------------------------- |
| **Stacks**               | Laravel (Filament, Voyager), Django, FastAPI, React/Vite e informes Markdown/LaTeX |
| **Agentes IA**           | No solo Cursor: adaptadores para Claude, Codex y Copilot                           |
| **CLI**                  | Comandos `validate`, `backlog`, `spec new`, métricas y más (Python 3.10+)          |
| **Principiantes**        | Guía en 5 minutos (`concepts.md`), plantilla de spec simple, ejemplos no técnicos  |
| **Proyectos existentes** | Modo agente: instalar sin romper documentación previa                              |
| **Calidad**              | Guía de desarrollo sano, guardias contra antipatrones en el flujo del agente       |
| **Informes**             | Perfil dedicado a documentos PDF/DOCX (no solo aplicaciones web)                   |

## Cómo empezar

### Requisitos

- **Git** (obligatorio)
- **Python 3.10+** (recomendado, para la CLI)
- El stack de tu perfil (PHP, Node, etc.)
- Un agente de IA (recomendado)

### Instalación rápida

```powershell
git submodule add https://github.com/jcalistop/sdd-kit.git sdd-kit
.\sdd-kit\bootstrap\init-sdd.ps1 -Profile laravel-filament -Project "Mi App"
```

- **Proyecto nuevo:** puedes ejecutar `init-sdd` tú mismo.
- **Proyecto con código ya existente:** añade solo el submodule y pide al agente que adopte SDD (ver [INSTALL.md](https://github.com/jcalistop/sdd-kit/blob/main/INSTALL.md)).

Guía completa: [README](https://github.com/jcalistop/sdd-kit/blob/main/README.md) · [INSTALL](https://github.com/jcalistop/sdd-kit/blob/main/INSTALL.md) · [Conceptos en 5 min](https://github.com/jcalistop/sdd-kit/blob/main/core/concepts.md)
